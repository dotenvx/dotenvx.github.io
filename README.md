## Notes

* Secrets must be kept
* 1. Run anywhere - dotenvx run -- node index.js
* 2. Multiple environments - dotenvx run -f .env.production
* 3. Encrypt - dotenvx encrypt
* 4. Keys off computer

Alternative

* 1. dotenvx encrypt
* 2. set DOTENV_PRIVATE_KEY in prod
* 3. run dotenvx run -- node index.js

More secure

* 1. dotenvx login
* 2. Same commands
* 3. KEYS OFF COMPUTER
